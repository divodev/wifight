# WiFight ISP Billing & Management System

Multi-vendor ISP billing system with support for MikroTik, Omada, Ruijie, and Cisco Meraki controllers.

## Features
- Multi-vendor controller integration
- RADIUS authentication & accounting
- Automated billing & subscriptions
- User self-service portal
- Admin dashboard
- Voucher generation
- Real-time session monitoring
